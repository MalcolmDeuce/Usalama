Usalama Visitor Check-In System

The Usalama Visitor Gatepass Management System is a system that ensures security in any organization by ensuring that visitors' details are recorded in detail upon any visit to the organisation.

The details that will be captured by the system include:

  • Visitor Name
  • Visitor ID/Passport Number
  • Time of check-in
  • Time of check-out
  • Visitor Phone Number
